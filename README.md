Aplikasi ini ditujukan sebagai aplikasi pemutar musik sederhana dengan fitur-fitur yang telah diinstruksikan.
Menggunakan MLL untuk list library (admin) dan daftar playlist (user), dan DLL untuk element lagu. Serta terdapat penggunaan stack untuk menyimpan data pemutaran laggu.
Lagu di dalam library dikelompokkan berdasarkan nama artis. Terdapat playlist favorit yang ototmatis ada saat daftarplaylist dibuat.
Pengguna akan diarahkan untuk masuk ke menu utama dan login sebagai admin/user.
- Jika login sebagai admin, masukkan password: playDeSongAdm
- Jika login sebagai user, masukkan password: musicGeeks

Fitur Admin:
1. Add New Singer -> admin dapat menambahkan artis baru ke dalam listLibrary.
2. Add New Song to Singer -> admin dapat menambahkan lagu baru ke dalam listLibrary, dan dikelompokkan berdasarkan nama penyanyi.
3. Delete song -> admin dapat menghapus lagu di dalam library, dan otomatis terhapus juga di seluruh playlist user.
4. Update Song -> admin dapat meng-edit lagu di dalam library, dan otomatis ter-edit juga di seluruh playlist user.

Fitur User:
1. Display Favourite Songs -> User dapat melihat list lagu yang sudah di-favoritkan.
2. Display a Playlist -> User dapat melihat list lagu yang ada di dalam sebuah playlist.
3. Display all song from library -> User dapat melihat semua lagu dari library.
4. Search your playlist -> User dapat mencari playlist dengan menggunakan kode playlistnya.
5. Search Song -> User dapat mencari lagu dari library, dan diberikan pilihan menu (playSong, add/remove favourite)
6. Edit playlist -> User diberikan pilihan untuk menghapus, menambah lagu, menghapus lagu, dan menambah playlist.
7. Display All Songs from A singer -> User dapat melihat list seluruh lagu dari seorang artis.
8. View History and Listening Time -> User bisa mengetahui sudah mendengar lagu apa saja, dan total waktu mendengarkan (menit dan detik)
